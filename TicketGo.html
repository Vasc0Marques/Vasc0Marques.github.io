Ticket Go

Site de venda de bilhetes para diversos eventos e espetáculos.

Link Do Projeto

Detalhes do Projeto

Este projeto consiste numa aplicação web de venda de bilhetes para eventos e espetáculos, desenvolvida com o framework Yii2 Advanced. A aplicação está dividida em backend e frontend, com uma base de dados MySQL para o armazenamento das informações. O objetivo é permitir que administradores gerenciem eventos, locais e bilhetes, enquanto os clientes podem consultar os eventos, e comprar bilhetes escolhendo as zonas.

O backend do sistema permite ao administrador gerenciar eventos, categorias, locais e zonas, além de definir preços para os bilhetes. Ele utiliza uma estrutura master-detail para criar zonas dentro dos locais e associar bilhetes aos eventos, com a possibilidade de escolher o preço e a zona correspondente. Todos os dados são armazenados numa base de dados MySQL.

❮ ❯



O frontend permite que os clientes visualizem a lista de eventos disponíveis, com detalhes como descrição das zonas e o respetivo preço. Após selecionar um evento, o cliente escolhe a zona desejada. Por fim, é possível adicionar bilhetes ao carrinho e concluir a compra diretamente na plataforma. Apos a compra é gerada uma fatura em pdf com os dados do cliente e os bilhetes comprados.
